# 🎮 Tic Tac Toe

A responsive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**. Play against a friend in a simple, interactive interface with automatic win and draw detection.

---

## 📌 Overview

This project is a browser-based implementation of the classic Tic Tac Toe game. It demonstrates core JavaScript concepts such as DOM manipulation, event handling, game logic, and state management while providing a clean and responsive user interface.

---

## ✨ Features

* 🎯 Two-player gameplay
* ✅ Automatic win detection
* 🤝 Draw detection
* 🔄 Restart game functionality
* 📱 Responsive design
* 🎨 Clean and user-friendly interface

## 🎮 How to Play

1. The game starts with Player **X**.
2. Players take turns clicking an empty cell.
3. The first player to align three marks horizontally, vertically, or diagonally wins.
4. If all cells are filled without a winner, the game ends in a draw.
5. Click the **Restart** button to play again.
